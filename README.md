# Pixel Dungeon

Roguelike-игра на Godot 4.5, созданная по туториалу канала "Студия Игор".

## О проекте

Этот проект — результат прохождения 2-й части туториала по созданию игры на движке Godot от YouTube-канала [Студия Игор](https://www.youtube.com/@studioigorofficial).

## Технологии

- Godot 4.5
- GDScript

## Запуск

1. Установите [Godot 4](https://godotengine.org/download)
2. Откройте проект через `project.godot`
3. Нажмите F5 для запуска
